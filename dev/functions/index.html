Helper function to find consumption at the constraint. Returns the residual the budget constraint.

  • c - consumption
  • j - state index
  • b1 - bond holdings today
  • b2 - bond holdings tomorrow
  • r - interest rate
  • θ - state value array
  • fac - parameter
  • gameta - parameter

Return

  • F - residual budget constraint
